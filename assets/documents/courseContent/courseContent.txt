HTML
HTML是一種用於網頁的標準標記語言，透過瀏覽器將內容呈現給使用者。HTML元素是構建網站的基石。

------
CSS
CSS 是一種樣式規則的語言，作用是設定各種樣式，例如不同字型、字體大小、間距、文字和背景顏色等等。

------
JavaScript
JavaScript是一種主要用於網頁前端的程式語言，可以增加與使用者間的互動，讓網頁動態更新內容，還可以成為非同步動態網頁。

------
Bootstrap
Bootstrap是目前網頁及行動裝置最知名的前端框架，內容包括HTML、CSS及JS等，它內含了RWD響應式網站設計功能。

------
Firebase
Firebase是一套雲端後端服務平台，提供行動裝置、網頁等多種開發情境所需要的功能，而且維護成本低廉。

------
RWD
RWD中文稱為「響應式網站設計」，是讓網頁在各種不同尺寸的裝置下，畫面都能合宜呈現的設計原則。
